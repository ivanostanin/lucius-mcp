# Contributing to Lucius MCP

Thank you for your interest in contributing to Lucius MCP for Allure TestOps!

## üöÄ How to Contribute

1. **Fork the repository** and create your branch from `main`.
2. **Review the architecture**: Familiarize yourself with the "Thin Tool / Fat Service" pattern
   in [docs/architecture.md](docs/architecture.md).
3. **Set up your environment**: Follow the [Setup Guide](docs/setup.md).
4. **Follow the Development Guide**: Use the instructions in [docs/development.md](docs/development.md) to add tools or
   services.
5. **Ensure quality**: Run linting, type checks, and tests before submitting a PR.
6. **Submit a Pull Request**: Provide a clear description of your changes and reference any relevant issues or stories.
